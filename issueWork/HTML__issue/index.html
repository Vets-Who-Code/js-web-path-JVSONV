This is a great website to learn CSS Grid in a fun and interactive way. Now only is it a great way to learn as a beginner, but it is also a great refresher.

As you learn you may come a cross template websites with paragraph's start like this: Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. You might think that this just looks like random gibberish. Could you believe that it is actually the work of a 15th century typesetter who scrambled parts of Cicero's De Finibus in order to provide placeholder text to mockup various fonts for a type specimen book!
